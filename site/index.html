welcome! this is a bunch of information about insertdisc5's game In Stars and Time, and its prequel, START AGAIN START AGAIN START AGAIN: a prologue.

this site will not avoid nor warn for spoilers for either game. i recommend completing ISAT and knowing about SASASAP as well as all optional content in both games before exploring this site.

what kinds of information?

this is essentially trying to be some sort of documentation on ISAT and SASASAP, in a way. things like variables and items and style guides and visuals and that stuff. the lists page has lists for variables, items, skills, and more, formatting will likely have things about how dialogue is formatted, visuals will have information about how to recreate visuals from the game, etc.

there's a lot of information about ISAT especially that i've gotten both out of my own curiosity and from working on the ISAT Script Project, some of which has been incredibly painful to figure out. a lot of reverse engineering, digging through code, and trial and error. i started making some references for myself to use, then realized that other people would probably benefit from it. for a while it was a series of markdown files on the ISATSP repository, then was moved to its wiki, and then i realized how incredibly painful github wikis are to work with and decided to move it to its own site. which is here! helo! waves!!!


other projects you should check out

  • felikatze's ISAT Script Project is amazing and i've helped a lot with it myself! go check it out, he's compiled a lot of dialogue from the game, and as he says this site is his baby. go look at it it's great
  • kongkrog's isat profile customizer does a great job at recreating visuals from the game, in both html/css and javascript generated images and gifs (it's seriously really cool, i would make something like this if i knew how to do javascript in this way lol)
  • the ISATwiki! if you're looking for story information, gameplay information, anything that isn't the tiny specific details in the game's code that make it work, go here. tons of people work on it and it's a good source of information about the game, the kind that most people actually care about
